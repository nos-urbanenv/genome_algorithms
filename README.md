# 0から始める環境ゲノム配列解析
ゲノム配列解析アルゴリズムに関する備忘録的まとめ

## 本稿執筆の動機
都市工学科に進学してから、情報解析と実験を両立できる研究分野をずっと探していた。
学部3年の春にふと、ゲノム情報解析に関わる研究であれば、その理想に近づけるのではないかと考え、農学部の入門的な講義をまず履修してみた。その講義をきっかけにバイオインフォにハマったため、生物情報科学の専門課程の講義を受けてみようと考えた。ちょうど良いタイミングでS2タームから理学部生物情報科学科の「ゲノム配列解析論Ⅰ」が開講されているのをシラバスで発見し、初めて扱う確率モデルやアルゴリズムとの新鮮な出会いを楽しみつつ、何とか単位も取得することができた。
この講義を通じて配列解析をアルゴリズム・計算機科学のレベルから考えることの面白さを知ることができ、大学3年の夏休みには古典的名著であるDurbin et.al(1998) "Biological Sequence Analysis"を読んでみたり、誰にも相談しようがないのでChatGPTを壁打ち相手に、アルゴリズムの理解が環境工学にもたらすであろういろいろな可能性について妄想してみたりしていた。

このように我流ながらも勉強を続ける中で、バイオインフォマティクス分野における書籍やネット上のリソースには、研究の実務とアルゴリズムレベルでの本質的理解を両立させようとする教材がほとんど存在しないことに気がついた。バイオインフォマティクス分野においては現在、様々なソフトウェアやパッケージ、解析環境が整ってはいるものの、それらの根幹となる数理・アルゴリズムに関しては、データ量が増えるにしたがって計算量が爆発的に増加するという動的計画法の性質などが原因で、未だ完璧なものとはいえない。そのような中、ブラックボックス化したソフトウェアを用いることには、少なからず危険が伴うはずだ。

私自身が卒論・修論でNGS解析を扱うかどうかはわからない。しかし、NGS解析は活性汚泥法、バイオレメディエーション、下水疫学など、今や環境工学の至る所で主要な解析手段として用いられている。解析の中身が理解できるだけでも、最新の研究動向への理解においては大きなアドバンテージとなるだろう。そこで本稿では、学習のアウトプットを兼ねて、「ゲノム解析アルゴリズム」と「実験系ラボでの実際の研究」について、学習者の視点から知識を再構成することを目標としたい。
